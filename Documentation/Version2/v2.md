- # Construction
	- ## Setup Instructions
		- Clone the repository
			-
			  ```shell
			  git clone https://github.com/yourusername/Go_Filer.git
			  cd Go_Filer
			  go mod init Go_Filer
			  ```
		- Directory Structure
			-
			  ```textile
			  Go_Filer/
			  ├── main.go
			  ├── go.mod
			  ├── Makefile
			  └── test/
			      ├── subfolder1/
			      │   ├── test1.go
			      │   ├── test2.go
			      ├── subfolder2/
			      │   ├── test3.go
			      │   ├── test4.go
			      └── ... (more subfolders and test files)
			  
			  ```
		- Build and run
			- Build the application
				-
				  ```shell
				  make build
				  ```
			- Run the application
				-
				  ```shell
				  make run
				  ```
	- ## Testing
		- ### Testing on windows
			-
			  ```shell
			  make test_wi
			  ```
		- ### Testing on Linux
			-
			  ```shell
			  make test_li
			  ```
